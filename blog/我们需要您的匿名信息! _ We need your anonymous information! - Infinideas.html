我们需要您的匿名信息! | We need your anonymous information!

  • 作者: Null
  • 时间: November 12, 2016
  • 分类: 默认分类

我们需要您上传一份匿名的显卡数据统计文件，来让我们的游戏开发者进一步知道我们玩家群体的电脑配置，以选择合适的函数来加快游戏运行速度。
We will be pleased if you can upload an anonymous statistical data which can make our game better.

如果您的操作系统为Windows并且想帮助我们的话，请下载附件NEWorld Launcher 161022.zip，并按如下步骤操作：

  1. 运行NEWorldLauncher.exe (实际上暂时还并不能启动用qwq)
  2. 在弹出的提示框内选择“是”
  3. 等待“成功上传统计数据！感谢您的支持”的提示，然后就可以关掉启动器辣！

提示: 本功能需要网络，并且可能耗费您合计约400KB的流量。

If your OS is Windows and you would like to help us, please download the attachment NEWorld Launcher 161022.zip and do the following operation.

  1. Run NEWorldLauncher.exe (which cannot launcher our game yet :) )
  2. Select "Yes" in the pop-up prompt box.
  3. Wait for the prompt box that contains "成功上传统计数据！感谢您的支持" (Upload statistical data successfully, thanks for your kind help.)

Tip: this operation need network and may cost your about 400KB data if you are using a metered network.
